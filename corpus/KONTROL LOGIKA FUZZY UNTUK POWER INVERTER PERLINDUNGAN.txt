Dalam penelitian ini kontrol fuzzy logic diimplementasikan pada parameter input untuk mengontrol pengisian tegangan input baterai yang disediakan pada sistem. Dengan menggunakan sistem logika rule fuzzy berdasarkan, baterai ditandai untuk mengidentifikasi parameter sinyal kebutuhan untuk dikenakan oleh sumber daya eksternal. Sebuah biaya PWM controller sirkuit yang digunakan, dan menggunakan bahasa pemrograman C, algoritma dari aturan fuzzy logic berdasarkan tertanam di papan kompatibel Arduino. Biaya PWM controller digunakan pada sistem ini untuk membangun baterai yang efisien pengisian untuk sistem.

A. Fuzzification Parameter Input.
The fuzzification parameter input didirikan berdasarkan karakterisasi dilakukan pada power inverter. Dua parameter input tegangan input inverter dan masukan inverter saat ini. Terlihat gambar 11 adalah fungsi keanggotaan untuk tegangan input inverter, di mana data untuk tegangan input disajikan di formulir persentase menggunakan tegangan baterai dasar dari 12 Volts. Para anggota fuzzy terdiri dari Kritis Rendah (CL), Sangat Rendah (VL), Low (L), Normal (N), Tinggi (H), Very High (VH) dan Kritis Tinggi (CH).

Untuk fungsi keanggotaan angka 13 adalah untuk pengisian PWM atau output defuzzifikasi dari kontroler logika fuzzy. Setiap anggota disajikan dengan persentase siklus PWM untuk pengisian baterai. Para anggota kabur untuk pengisian PWM terdiri dari None Charging, Charging Lambat, Float Pengisian, Charging Nominal, Charging Cepat, Pengisian cepat dan Meningkatkan Pengisian.

B. Inference Fuzzy menggunakan metode Mamdani untuk Power
Inverter Perlindungan Menggunakan perangkat lunak MATLAB metode Mamdani ini digunakan untuk memproses metode berbasis aturan fuzzy pada perlindungan power inverter.
The dual input single output dilihat pada gambar 16 ini dibuat dengan menggunakan MATLAB. Daerah kuning menunjukkan kapan pengisian meningkatkan PWM diperlukan karena tegangan input rendah dan arus masukan. Di sisi lain, area hijau menunjukkan ketika nominal pengisian PWM diperlukan di daerah dekat tingkat biaya yang ideal. Terakhir, area biru menunjukkan, pengisian lambat ketika tegangan input dan arus masukan tinggi.
